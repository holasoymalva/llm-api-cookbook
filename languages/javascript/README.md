# JavaScript / TypeScript Examples

## Prerequisites

-   Node.js (v18+ recommended)
-   npm or pnpm/yarn

## Installation

Run `npm install` inside this directory (if we had a package.json here) or generally just create a `package.json` at the root for convenience.

Typically, for these examples, you will need:

```bash
npm install openai groq-sdk dotenv
```

To run TypeScript examples directly:

```bash
npx ts-node <path-to-file>
```
